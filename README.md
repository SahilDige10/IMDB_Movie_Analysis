## ğŸ”‘ Key Insights
1.  **The Runtime Sweet Spot:** The ideal runtime for a top-rated movie is **100-130 minutes**.
2.  **Volume vs. Value:** While **Drama** is the most common genre, **Adventure & Animation** yield the highest average ROI.
